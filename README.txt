This is the total project of the mips sembler,which has a graphics interface.
In the 设计报告.pdf,there is some information about how to use the sembler.
In the excutable_file,there has a mips_zxm.exe,which can be run directly.
In the bin_and_coe folder,there are bin and coe files ,which can be loaded into the sembler.
In the source_code folder, there are all my cpp file,header file and the qt project.
In the assembler_test.txt and disassembler_test.txt,I provide some test examples,which can be open and load to the sembler.
